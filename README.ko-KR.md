# Lumeflow Skill Hub

OpenClaw 플랫폼을 위한 AI Agent 스킬 패키지. 자연어로 원하는 것을 설명하기만 하면 전문적인 영상과 이미지를 생성합니다 — 수동 조작 불필요, 학습 곡선 제로.

이 스킬은 Agent Skills 사양을 준수하며, 호환되는 AI 코딩 어시스턴트(Cursor, Claude Code 등)와 함께 사용할 수 있습니다.

**언어**: [English](README.md) | [中文](README.zh-CN.md) | [日本語](README.ja-JP.md) | [한국어](README.ko-KR.md)

## 무엇을 할 수 있나요?

스킬을 설치한 후, AI에게 필요한 것을 말하기만 하면 됩니다. 다음은 몇 가지 예시입니다:

### 한 문장으로 창작

| 당신이 말하면…… | 얻게 되는 것…… |
|----------------|---------------|
| "5장의 서로 다른 스타일의 제품 사진을 생성해줘" | AI가 생성한 5장의 이미지, 배치 병렬 생성 |
| "석양에 해변을 걷는 고양이 10초 영상을 만들어줘" | AI가 생성한 영상 클립 |
| "이 사진으로 영상을 만들어줘" | 이미지를 참고한 영상 생성 |
| "이 물병을 패션 모델에게 들려줘" | 모델이 제품을 들고 있는 합성 이미지 |
| "주황색 고양이를 그려줘, 9:16 세로" | AI가 생성한 이미지 |
| "이 제품 사진의 배경을 제거해줘" | 깔끔한 컷아웃 PNG |
| "이 사진의 배경을 트로피컬 해변으로 바꿔줘" | AI가 편집한 이미지 |
| "이 참고 영상의 스타일을 따라 새 영상을 만들어줘" | 스타일 전이 영상 |

### 사용 방법

OpenClaw에서 AI Agent와 자연어로 대화하면 됩니다:

- **영상 생성** — "고양이가 주방에서 요리하는 영상을 만들어줘, Pixverse C1으로, 720P, 5초"
- **이미지 생성** — "GPT Image 2로 주황색 고양이를 그려줘, 9:16 세로, 2K 해상도"
- **이미지→영상 / 이미지→이미지** — 참조 이미지를 전송하거나 설명하면 Agent가 자동으로 업로드하여 입력으로 사용
- **잔액 확인** — "크레딧 잔액이 얼마나 남았어?"
- **이력 조회** — "이전 생성 작업 보여줘"

생성 전에 Agent가 파라미터 요약과 예상 크레딧 소비량을 표시하며, 확인 후에만 작업을 제출합니다.

**상상력이 유일한 제한입니다** — 위의 예시는 시작점에 불과합니다. 이미지 생성, 영상 제작 등을 자유롭게 조합하여 상상할 수 있는 모든 워크플로우를 구축할 수 있습니다. 자신의 언어로 창의적인 비전을 설명하면 AI가 실현 방법을 찾아냅니다.

## 프로젝트 구조

```
lumeflow-video/
├── SKILL.md                  # 스킬 매니페스트 (v1.0.0)
├── capabilities/             # 기능 정의 (워크플로우, 규칙, 모델 파라미터)
│   ├── workflow.md           # 필수 7단계 워크플로우
│   ├── rules.md              # 출력 형식 및 대화 규칙
│   ├── video.md              # 영상 모델 및 파라미터 제약
│   ├── image.md              # 이미지 모델 및 파라미터 제약
│   ├── request-templates.md  # JSON 요청 템플릿
│   ├── result-handling.md    # 결과 처리 및 자동 재시도
│   ├── parameter-flow.md     # 파라미터 수집 플로우
│   └── upload.md             # 파일 업로드 형식 및 media:// 프로토콜
├── references/
│   ├── api_reference.md      # API 엔드포인트 참조 및 오류 코드
│   └── ai_models_analysis.md # 모델 파라미터 심층 분석
└── scripts/
    ├── generate.py           # 메인 진입 스크립트 (생성, 조회, 업로드, 잔액)
    ├── login.py              # 브라우저 로그인 플로우
    ├── sign.py               # API 요청 서명 (MD5)
    ├── write_body.py         # JSON 본문 작성기 (UTF-8 BOM 없음)
    ├── task_store.py         # 로컬 작업 저장소
    └── product_config.py     # 런타임 설정 리졸버
```
